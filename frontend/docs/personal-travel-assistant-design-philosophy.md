# Star Atlas

这套视觉语言把空间当成首要信息媒介，而不是文字的陪衬。界面必须留出大面积的静默黑场，让功能像被聚光灯挑出的星点一样浮起；每一个圆角、留白和模块边界都要像被反复推敲过很多轮，呈现出经过长时间雕琢后的克制秩序。它不是热闹的科技感，而是被精密校准后的深色静场，要求极高完成度与极稳的节奏。

色彩应当只保留极少数层级：近黑底色承载纵深，冷灰负责体积，唯一的冷蓝作为方向性的信号。蓝色不是装饰，而是节奏节点，只在品牌、激活态和关键动作上短暂发光。所有亮度差都应经过细密打磨，避免任何廉价的高饱和跳色，让画面看起来像出自深厚经验之手，经过了耐心而讲究的明度控制。

比例与尺度要显得果断。左侧历史区像目录索引，密度高却不噪；中间主舞台必须宽阔、留白充足，让标题、模式切换和输入容器拥有清晰的呼吸关系。重要信息用更大的体量与更慢的节奏出现，次要信息退到边缘，以一种近乎编辑设计的方式建立层级。这种层级感必须看上去经过大量人工微调，而不是自动排布的结果。

材质表现强调“玻璃并不透明，金属也不冰冷”的边界感。模块表面应该有极轻的内描边、雾化阴影和克制的高光，让容器像被打磨过的深色树脂，而不是普通卡片。任何按钮、胶囊标签和对话气泡都要保持触感统一，呈现出精工细作后的连续性，让人一眼就感觉到这是经过耐心修边和反复校准的高完成度作品。

文字必须节制，只承担锚点作用。标题可以锋利，说明文字必须退后，按钮文案则要像界面雕刻的一部分存在。整套作品应始终保留一种“被精密绘制过”的气质，看起来像由领域顶尖的人花了很长时间慢慢磨出来，细节里带着明确判断，整体上则安静、稳定、几乎没有多余动作。
