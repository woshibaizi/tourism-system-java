# A*算法实现总结

## 📋 项目概述

本项目成功将旅游系统中的Dijkstra最短路径算法替换为A*算法，在保持向后兼容性的同时，提升了路径搜索的效率。

## 🔧 实现细节

### 1. 数据结构
```python
# 优先队列：(f_score, 节点)
open_set = [(f_score[start], start)]

# 评分系统
g_score = {node: float('inf') for node in self.graph}  # 实际代价
f_score = {node: float('inf') for node in self.graph}  # 总评分
```

### 2. 核心算法流程
1. 初始化起点的g_score和f_score
2. 将起点加入open_set
3. 循环直到找到目标或open_set为空：
   - 选择f_score最小的节点
   - 如果是目标节点，重构路径
   - 否则扩展邻居节点，更新评分
